# 小红书文案提示词模板

## 适用场景

- AI 工具推荐
- 教程笔记
- 使用体验分享
- 工具对比
- 效率方法总结

## 基础提示词

请你作为一名小红书内容编辑，帮我写一篇小红书笔记。

主题：【填写主题】
目标读者：【填写读者，比如 AI 新手、职场人、自媒体作者】
内容目标：【种草 / 教程 / 对比 / 避坑 / 总结】
语气风格：【自然、口语化、不要夸张】
字数：【填写字数】

请输出：

1. 5 个标题备选
2. 正文初稿
3. 适合配图的 4 个画面建议
4. 8 个标签
5. 需要人工核对的信息

## 追问优化

请基于刚才的笔记继续优化：

1. 开头更直接，前 3 秒能吸引读者
2. 删除空泛表达
3. 增加一个真实使用场景
4. 结尾加一个自然的收藏或评论引导

## 检查清单

- 标题是否说明具体收益
- 正文是否有真实场景
- 是否避免夸大承诺
- 是否需要补充来源或截图
- 标签是否和内容相关
